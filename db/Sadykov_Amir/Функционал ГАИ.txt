Функционал:
1. Авторизация.
2. Водители:
 - добавление новых водителей.
 - продление водительских прав.
 - удалить водителя из базы.
 - поиск водителя(по номеру прав, по ФИО,...).
3. ТС:
 - добавление ТС(регистрация).
 - удаление.
 - поиск ТС(по VIN,по рег. знаку, по владельцу, по номеру регистрации,...).
 - добавление ТС в список угнанных.
 - удаление ТС из угнанных.
 - показать все угнанные авто.
 - поиск ТС, у которых закончилось ТО.
 - сообщить водителю, что ТО закончилось.
 - установить дату след. ТО для ТС.
4. Штрафы:
 - поиск штрафов(по номеру ТС, по номеру протокола).
 - показать все штрафы.(по времени, по ТС, по типу штрафа,...).
 - добавление.
 - удаление.
