База данных "Гостиница"

Для посетителей доступны:
  - регистрация в системе
  - вход в систему по логину-паролю
  - каталог номеров; поиск подходящего номера
  - бронирование номера (свободного и незабронированного)
  - вывод своей истории посещений
  - просмотр и редактирование личных данных
Для работника доступны:
  - вход в систему по логину-паролю
  - вывод информации по занятым комнатам; поиск
  - поиск свободных комнат
  - история посещений; поиск в истории
  - освобождение комнаты (при этом посетителю насчисляется балл)
  - учет посетителей; поиск
  - добавление посетителя в базу
  - заселение посетителя в номер
  - фиксирование оказанных услуг
  - расчет стоимости комнаты с учетом накопленных посетителем баллов (балл за каждое посещение)
  - расчет дополнительной стоимости пребывания в номере с учетом оказанных услуг
  - снятие брони с номеров
