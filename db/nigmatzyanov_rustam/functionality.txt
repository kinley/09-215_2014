1. Для преподавателя доступны:
- вход в систему по логину-паролю
- добавить студента, сдающего комплекс ГТО
- добавить результат сдачи комлпекса ГТО
- посмотреть список испытаний и соответствующие нормативы
- посмотреть информацию о студентах
- посмотреть результат сдачи комплекса ГТО отдельным студентом
2. Для студента доступны:
- вход в систему по логину-паролю
- посмотреть список испытаний и соответствующие нормативы
- посмотреть информацию о себе 
- посмотреть свой результат сдачи комплекса ГТО
3. Для гостя доступны:
- вход в систему без регистрации
- посмотреть список испытаний и соответствующие нормативы
