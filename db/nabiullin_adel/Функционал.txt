База данных "Помощник в наборе команды для геймдева"

Для пользователей доступны:
  -регистрация в системе
  -вход в систему с использованием логина и пароля
  -каталог объявлений о поиске человека в команду на определенную должность
  -поиск необходимой вакансии по каталогу с использованием чекбоксов для критериев поиска
  -подача запросов на прием в команду
  -принятие/отклонение запросов на вступление в команду, присланных тимлидами
  -самовольный выход из команды
  -создание резюме(не больше 5 штук единовременно)
  -просмотр публичных контактных данных других пользователей, включая все их резюме
  -просмотр и редактирование личных данных
  -создание ниши проекта для набора в нее команды, проекту дается имя, а создатель автоматически становится тимлидером
Плюс к этому для Тимлидов доступны:
  -подача запросов пользователям с приглашением вступить в команду
  -принятие/отклонение запросов, присланных пользователями
  -изгнание тиммейта из команды
  -размещение объявления о поиске человека в команду с указанием должности
