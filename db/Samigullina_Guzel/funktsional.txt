База данных "Пекарня"
Для заказчика доступны:
- вход в систему по логину-паролю
- просмотр меню
- заказ еды
- информация о сделанном заказе
- просмотр и редактирование личных данных (ФИО, телефон, адрес)

Для пекаря:
- вход в систему по логину-паролю
- добавление заказчика в базу
- вывод информации по заказчикам
- приём заказа
- вывод информации по заказам
- редактирование статуса заказа
- просмотр и редактирование меню

Для курьера:
- вход в систему по логину-паролю
- вывод информации по заказчикам
- вывод информации по заказам
